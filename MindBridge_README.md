# MindBridge — Mental Health Peer Support App & Responsive Website

**Platform:** iOS + Responsive Web &nbsp;|&nbsp; **Tools:** Figma + HTML/CSS/JS &nbsp;|&nbsp; **Status:** Hi-fi + Coded &nbsp;|&nbsp; **Type:** Social Good

---

## Overview

MindBridge is a mobile app, responsive website, and fully coded interactive prototype filling the critical gap between *"I'm not okay"* and *"I need a therapist."* It provides anonymous, free peer support through moderated circles, flexible mood tracking, and always-visible crisis resources.

AI tools (Claude, ChatGPT) were integrated throughout design and research. A fully coded HTML/CSS/JS website was built across all 12 screens with interactive components throughout.

---

## The Problem

- Pre-clinical users experiencing real anxiety or burnout with no peer support alternative
- Caregivers — highest-burnout group, almost invisible in consumer mental health design
- High-stigma communities where cultural barriers prevent formal help-seeking

---

## The Solution

| Pillar | Description |
|---|---|
| Anonymous by default | No email required — username and optional avatar only |
| Moderated peer circles | Topic-based communities with trained peer supporters |
| Flexible mood tracking | Any cadence, no streaks, no daily obligation |
| Always-visible crisis support | Persistent in global navigation — never buried |
| Peer supporter badge pathway | Structured, safe giving-back with recognition |
| AI circle matching | NLP maps onboarding description to relevant circles |

---

## Coded Prototype Features

- 12 fully navigable screens with persistent sidebar navigation
- Interactive mood emoji selector with SVG history chart
- Animated box breathing exercise with 4-phase progression
- Live peer circle chat with real message sending
- Working self-care task checklist with completion animations
- Crisis resource severity selector with contextual content
- Toggle switches for all notification preferences
- Sign up with name persistence to dashboard greeting

---

## AI Integration

### In the Design Process

- Claude used to synthesise academic literature on digital mental health
- AI competitive gap analysis across 5 competitors and 14 criteria
- AI identification of edge case personas (David — high-stigma older male; Sophie — hospice nurse)

### In the Product

- NLP circle matching at onboarding — removes self-diagnosis requirement
- Passive distress signal detection for moderator escalation
- Mood pattern insights surfacing correlations in logged data
